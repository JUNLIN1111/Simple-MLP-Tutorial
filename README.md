# 🌸 Kawaii MLP Actor – Tiny Cute Neural Network Baby! 🤖💖

A super-duper simple **Multi-Layer Perceptron** made with just NumPy~!  
No scary big libraries — perfect for little learners who want to see how a baby AI brain thinks! 🧠✨

## ✨ Features of Our Adorable Friend

- 3 cozy layers (input → hidden hug → hidden hug → output)
- Uses `tanh` squishy activation → outputs wiggle between -1 and 1 ♡
- Hand-written backpropagation (we see every little learning step!)
- Batch training so our baby can learn from many examples at once 💕
- Only needs NumPy — like needing just milk and cuddles!
